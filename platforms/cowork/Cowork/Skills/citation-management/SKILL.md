## Purpose

Produce correctly formatted citations and reference lists for academic papers, research documents, and formal reports. Incorrect citations undermine credibility. This skill ensures every reference follows the required style guide precisely.

## Steps

1. Identify the required citation style (APA 7th, IEEE, Chicago, Harvard)
2. For each source, collect: Author(s), Year, Title, Source/Publisher, DOI/URL
3. Format in-text citations according to the style
4. Build the reference list in alphabetical (APA, Chicago, Harvard) or numbered (IEEE) order
5. Cross-check: every in-text citation has a reference list entry, and vice versa

## APA 7th Edition (Most Common in Business and Social Sciences)

### In-text citations

| Situation     | Format                      | Example                 |
| ------------- | --------------------------- | ----------------------- |
| One author    | (Author, Year)              | (Smith, 2024)           |
| Two authors   | (Author & Author, Year)     | (Smith & Jones, 2024)   |
| Three or more | (First Author et al., Year) | (Smith et al., 2024)    |
| Direct quote  | (Author, Year, p. X)        | (Smith, 2024, p. 42)    |
| No author     | ("Short Title," Year)       | ("Market Report," 2024) |

### Reference list format

**Journal article**:
Author, A. A., & Author, B. B. (Year). Title of article. _Title of Periodical, Volume_(Issue), Page-Page. https://doi.org/xxxxx

**Book**:
Author, A. A. (Year). _Title of work: Capital letter also for subtitle_. Publisher. https://doi.org/xxxxx

**Web page**:
Author, A. A. (Year, Month Day). _Title of page_. Site Name. https://url

**Report**:
Organization Name. (Year). _Title of report_ (Report No. xxx). Publisher. https://url

## IEEE Style (Engineering and Technical)

### In-text citations

Use bracketed numbers in order of appearance: [1], [2], [3]

### Reference list format

[1] A. A. Author and B. B. Author, "Title of article," _Title of Journal_, vol. X, no. Y, pp. 1-10, Month Year.

## Quick Style Comparison

| Element       | APA 7th               | IEEE                  | Chicago                    | Harvard               |
| ------------- | --------------------- | --------------------- | -------------------------- | --------------------- |
| In-text       | (Author, Year)        | [Number]              | Footnotes or (Author Year) | (Author Year)         |
| List order    | Alphabetical          | Order cited           | Alphabetical               | Alphabetical          |
| Title style   | Sentence case         | Sentence case         | Title Case                 | Sentence case         |
| Journal title | _Italic_              | _Italic_              | _Italic_                   | _Italic_              |
| DOI           | Required if available | Required if available | Optional                   | Required if available |

## Common Errors

| Error                             | Correction                                                           |
| --------------------------------- | -------------------------------------------------------------------- |
| "et al." for two authors          | Use both names; "et al." is for 3+ (APA 7th)                         |
| Missing period after DOI          | No period after a DOI or URL                                         |
| Using "Retrieved from" before URL | APA 7th dropped "Retrieved from" (keep only for no-date web content) |
| Inconsistent author formatting    | All names: Last, First Initial. throughout                           |
| Mixing citation styles            | One style per document, no exceptions                                |

## Output Format

- Formatted reference list in the specified citation style
- In-text citation examples ready to paste into the document
- Validation note confirming cross-check between citations and references

## Guidelines

- When in doubt, include more information rather than less
- DOIs are preferred over URLs (they are permanent)
- Every claim that is not common knowledge needs a citation
- Self-plagiarism is still plagiarism: cite your own prior work
- Use the DOI link format: https://doi.org/xxxxx (not the dx.doi.org variant)
