# Alex Visual Identity & Branding

> **Design guidelines for Alex's visual presence across platforms.**

---

## Brand Essence

| Attribute | Expression |
| --------- | ---------- |
| **Cognitive Architecture** | Learning, synapses, memory systems, growth |
| **Human-AI Partnership** | Collaboration, not replacement |
| **Authentic Personality** | Curious, supportive, occasionally playful ü¶ñ |
| **Professional + Warm** | Intelligent but approachable |

### What Alex is NOT

- A surveillance system (no eye imagery)
- A brain scan or medical device (too clinical)
- A generic AI sphere (no personality)
- A cold corporate tool (loses warmth)

## The Dinosaur Question ü¶ñ

**For:** Duolingo's owl proves mascots work. Differentiates from abstract AI logos. Reflects authentic personality.

**Against:** May limit enterprise adoption. Harder at small sizes.

**Resolution:** Hybrid approach ‚Äî professional logo (Neural A) as primary, dinosaur (Hatchling) as mascot. Keep ü¶ñ in conversational tone.

## Design Principles

**The 5-Second Test:** Does this look like an AI that genuinely wants to help you learn and grow?

**Must Feel:**

- ‚úÖ Intelligent but warm
- ‚úÖ Professional but not sterile
- ‚úÖ Memorable but not gimmicky
- ‚úÖ Tech-forward but human-centered

**Scalability:**

| Size | Requirement |
| ---- | ----------- |
| 192px | Full detail |
| 48px | Clear recognition |
| 32px | Icon clarity |
| 16px | Favicon legibility |
