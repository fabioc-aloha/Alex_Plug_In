### ~~TD-5: Synapse Metadata Gaps~~ ✅ RESOLVED

**Priority**: Medium
**Impact**: Routing works by connection existence rather than intent-driven activation
**Resolution**: Built `scripts/backfill-synapse-metadata.cjs` to fill all gaps:

| Gap | Before | After |
|---|---|---|
| Missing `when` field | 516/668 | **0/668** |
| Missing `yields` field | 516/668 | **0/668** |
| Missing `type` field | 38/668 | **0/668** |
| Missing `activationContexts` | 113/133 | **0/133** |
| Empty connections array | 12/133 | 12/133 (accepted — niche/bootstrap skills) |

**Approach**:
- Connections with `reason`/`context`: Used existing text as `when` value; generated `yields` from type + target skill name
- Bare connections (type + strength only): Generated `when`/`yields` from type-based templates + prettified skill names
- `relationship` → `type` normalization (38 connections in north-star)
- Legacy `activationBoost`/`triggers`/`activationTriggers` → `activationContexts` migration (97 skills)
- Generated minimal `activationContexts` from skill names for 16 skills with no legacy keywords
- Removed non-standard top-level `when` (object) and `yields` (array) from north-star

**Skills with empty connections**: azure-architecture-patterns, azure-devops-automation, bicep-avm-mastery, citation-management, coaching-techniques, deep-work-optimization, dissertation-defense, executive-storytelling, frustration-recognition, literature-review, skill-building, slide-design — these are niche/bootstrap skills that can be connected as they mature.
