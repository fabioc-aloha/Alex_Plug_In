# Copilot Cowork Research

Research collected: 2026-04-02
Status: Frontier Preview (available via Frontier program since March 30, 2026)

## What is Copilot Cowork?

Copilot Cowork is the execution layer for Microsoft 365 Copilot. It moves beyond single-turn prompts and responses toward long-running, multi-step work that unfolds over time. You describe the outcome you want, and Cowork breaks it down into a plan, executes steps across Microsoft 365 apps, and keeps you in control throughout.

Cowork was announced as part of **Wave 3 of Microsoft 365 Copilot** on March 9, 2026, and became available via the Frontier program on March 30, 2026.
