## Workshop Format

Each workshop includes:

1. **Concepts** — Key ideas and mental models
2. **Walkthrough** — Guided demonstration
3. **Exercises** — Hands-on practice
4. **Challenge** — Real-world scenario
5. **Reference** — Quick lookup card



## Fundamentals Track (2 hours)

### Module 1: Introduction (30 min)

- What is the GCX Copilot?
- The Four Pillars model
- How it enhances Copilot

### Module 2: Setup & Configuration (30 min)

- Prerequisites and installation
- VS Code settings
- Verifying your setup

### Module 3: Basic Usage (30 min)

- Asking effective questions
- Using slash commands
- Understanding responses

### Module 4: Best Practices (30 min)

- Providing context
- Choosing the right pillar
- Trust but verify
