# GCX Copilot

> Your intelligent partner for code, documentation, and enterprise integrations.

---

## What is Copilot?

The GCX Copilot is an AI-powered development assistant that helps you:

- **Write better code** — Reviews, refactoring, debugging, testing
- **Create documentation** — ADRs, specs, runbooks, guides
- **Manage knowledge** — Search, organize, synthesize information
- **Connect systems** — MS Graph, ADO, Fabric, CPM, Everest

---

## Quick Start

### 1. Open VS Code

Open your project in VS Code with GitHub Copilot enabled.

### 2. Start a Conversation

Press `Ctrl+Shift+I` (Windows) or `Cmd+Shift+I` (Mac) to open Copilot Chat.

### 3. Ask Anything

```
@workspace How do I add a new API endpoint?
```

```
@workspace Review this function for security issues
```

```
@workspace Write an ADR for our caching strategy
```
