## Usage in Extension

Agent mode banners can be referenced from `.github/agents/*.agent.md` files:

```markdown
<!-- In .github/agents/Researcher.agent.md -->
iconPath: alex_docs/alex3/agents/AGENT-RESEARCHER.png
```

Cognitive state portraits can be used in `src/views/welcomeView.ts` to show
Alex's current activity state contextually.

---

## Generation

```powershell
# Both series
$env:REPLICATE_API_TOKEN = "r8_..."
node scripts/generate-alex-agent-images.js

# Agent banners only
node scripts/generate-alex-agent-images.js --series=a

# State portraits only
node scripts/generate-alex-agent-images.js --series=b

# Dry run (preview prompts without charging)
node scripts/generate-alex-agent-images.js --dry-run
```

Report saved to `alex_docs/alex3/generation-report.json` after each run.
